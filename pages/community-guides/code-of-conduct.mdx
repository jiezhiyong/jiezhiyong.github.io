# 贡献者行为准则

贡献者遵守本行为准则，才能使社区成为开放、热情、多元化、包容的环境。

### 为什么要有行为准则

行为准则是​​为项目参与者建立行为期望的文档。采用并执行行为准则有助于为社区营造积极的贡献氛围。

### 适用范围

- 我们致力于为每个人提供一个友好、安全和热情的环境，无论年龄、体型、文化、种族、性别表达、性别认同、经验水平、国籍、个人能力或残疾、外貌、身体或心理差异、种族、宗教、技能、性取向、社会经济地位和亚文化。无论这些或其他属性如何，我们都欢迎人们。

- 我们在一起会更好。我们的相似之处多于不同之处。

- 我们的社区建立在相互尊重、宽容和鼓励的基础上。

- 我们相信，一个人们互相尊重的多元化社区会更强大、更有活力，并拥有更多潜在的贡献者和更多的想法来源。我们的目标是更加多元化。

- 我们对每个人都友善、热情、有礼貌。

- 我们尊重他人、他们的职位、他们的技能、他们的承诺和他们的努力。

- 无论是当面交流还是在线交流，我们都很专注，并且在表达不同观点时会很机智。

- 我们知道语言塑造现实。因此，我们在提供的文件中以及与人们交谈时使用包容性、性别中立的语言。当提到一群人时，我们的目标是使用中性术语，如“团队”、“人们”、“每个人”。

- 我们尊重人们的意见分歧并提出建设性的批评。

- 我们重视人胜过代码。

### 应当停止的行为

- 不要歧视任何人。

- 任何形式的性别歧视和种族主义（包括性别歧视和种族主义“笑话”）、贬低或侮辱行为以及骚扰均被视为直接违反本行为准则。骚扰包括与年龄、体型、文化、种族、性别表达、性别认同、经验水平、国籍、个人能力或残疾、外貌、身体或心理差异、种族、宗教、技能、性相关的冒犯性口头评论。取向、社会经济地位和亚文化。骚扰还包括公共场所的性图像、故意恐吓、跟踪、跟踪、骚扰性摄影或录音、不适当的身体接触以及不受欢迎的性关注。

- 在线或离线通信渠道上，请勿使用明显的性昵称或其他可能有损所有人友好、安全和热情环境的昵称。

- 不要刻薄或粗鲁。

- 尊重某些个人和文化认为随意使用脏话是冒犯和令人反感的。

- 在任何其他渠道进行不受欢迎/未经同意的性挑衅是不允许的。

- 语气争论或以其他方式利用人们友善的愿望是不受欢迎的，特别是在讨论违反本行为准则的情况时。

- 请避免无结构的批评。

- 同样，任何垃圾邮件、恶意攻击、煽动、诱饵或其他窃取注意力的行为都是不受欢迎的。

- 不要公布他人、公司或者组织的相关信息。

- 禁止恶搞、侮辱、贬损、评论、威胁、骚扰等文字或者图片。

- 禁止发布任何违反国家法律法规、涉政的内容。
